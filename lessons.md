## 🎯 KNOCKOUT SCORING — ADDITIVE, max 7 (the active system)
Scored on the **90/120-minute result**. **Penalties ignored** — the shootout winner earns nothing; a tie level after 120 is scored as a **DRAW**. Additive:
- **Correct outcome (W/D/L): +2** · **each team's goal count matched: +1** (→ +2) · **exact scoreline: +3 bonus** · **MAX = 7**.
- Only **0 / 1 / 2 / 3 / 7** are achievable (matching both goals = the exact = 7).
- _Worked ex. (actual 2-2, won on pens): 2-2→7, 1-1/0-0→2, 2-0→1, 1-3→0._

### What this changes vs the group stage (was 6/4/3/1/0)
1. **The exact (7) is hugely dominant — 7 vs next-best 3.** Nailing the single most-likely scoreline IS the game. Hunt it every match.
2. **A correct outcome alone is only 2** → don't play safe; commit to a scoreline.
3. **Goal-count matching pays independently (+1, even on a wrong outcome)** → always use realistic goal counts.
4. **DRAWS ARE LIVE** (a tie → penalties is scored as the 90/120 draw) → **predict 1-1 for cagey, evenly-matched ties** (common knockout result + 7 if exact). Never predict who wins on penalties.

## ⭐ NORTH STAR (carried forward)
1. **Outcome first, then scoreline.** Correct W/D/L is the gateway; the exact wins the pool.
2. **Back the bookmaker favorite** unless a concrete, verified reason says otherwise.
3. **Never 0-0 → 1-1.** Concentrate on **common knockout scores (1-0, 1-1, 2-1, 0-1, 2-0)**.
4. **X-0** for a clear favorite likely to keep a clean sheet.
5. **Commit the margin** — don't hedge (R1's scoreline leak was timidity).
6. **For two even, organized sides → predict the 1-1 draw.** Jun 28 proved we can call draws (ALG-AUT 3-3, COL-POR 0-0 both hit); knockouts have *more* draws (penalty-ties scored as draws). This was our biggest R1 fix — keep it.

## Knockout regime notes
- **No dead rubbers / content favorites / rotation** — every game win-or-out, both trying. R1's biggest leak (favorites coasting in low-stakes games) is gone.
- **Cagier, lower-scoring** → favor 1-0 / 2-1 / 1-1; fewer blowouts (but still back a clear margin in a real mismatch).
- **A genuine mismatch (group winner vs a 3rd-place qualifier)** → back the favorite with a clean decisive score (often X-0). **Two evenly-matched sides** → 1-1 is live.
